properti pada background:

  • background-colour: untuk mengatur warna pada bachground
  • background-image: untuk mengatur gambar yang akan digunakan pada background
  • backgroud-position: mengatur posisi gambar pada background
  • background-repeat: untuk mengatur jenis pengulangan gambar pada background

  • color: untuk memberikan warna
  • text-align :mengatur format paragraf/teks
  • text-indent: memberi identitas pada paragraf/teks
  • text-decoration: mengatur dekorasi pada teks
  • text-transform: mengubah jenis huruf menjadi huruf besar, kecil / kapital
  • letter-specing: mengatur spasi antar huruf
  • word-spacing: mengatur spasi antar kata

id

  • sebuah elemen html hanya boleh memiliki 1 id
  • setiap halaman hanya boleh memiliki 1 elemen dengan id tersebut
  • dapat digunakan sebagai penanda halaman untuk link
  • digunakan juga untuk js
  • sebaiknya tidak digunakan untuk css (lebih baik gunakan class)

pesudo-class yang berhubungan dengan link

  • :link style default pada sebuah link (a yang memiliki href)
  • :hofer style ketika kursor mouse berada diatas sebuah link/elemen
  • :active style ketika sebuah link di klik (keadaan aktif)
  • :visited style ketika sebuah link sudah pernah dikunjungi sebelumnya (berlaku pada browser yang sama)
