BOX-SIZING

jadi harus hati-hati, misalkan kita membuat sebuah kotak, lalu didalam nya dikasih sebuah padding dan border. maka dimensi kotak tersebut akan berubah ukuran nya sesuai ukuran tambahan dari pading atau bordernya. untuk mengatasi hal tersebut css punya properti khusus agar kita tidak perlu meng kalkulasi angka nya. properti tersebut dinamakan box-sizing


  • content-box : nilai default, dimana padding dan border akan menambah ukuran box
    • kotak ini berisi nilai content-box
